TTRPGs

A Table Top Role Playing Game is a game where a group of players create characers to act out, like an actor improvising a character. At every table the game is played differently, with house rules or home made systems, each campaign is it's own world that the players create together.

My Services

As a Game Master I introduce players to the system and the game world or campaign. I ensure the players learn the basics of how to engage with the game as well as working together with the other players, it would be difficult to play a sport alone and it's the same here.
